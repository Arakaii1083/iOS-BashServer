BashServer is a simple server, written in bash, that allows webapps to execute commands on an iOS device.

Consult the Examples Directory for examples on implementation.

BashServer is written by and maintained by @innoying if you have issues please contact me on twitter or via my email: luke@innoying.com

If you wish to experiment with this before it is added to a major default repository you may add http://innoying.com/repo to Cydia.
